# nr - Modern and pretty alternative of npm-run-all

## Prerequisites

- Node.js v20+

## Installation

```shell
npm install -D @siketyan/nr
```

or

```shell
pnpm add -D @siketyan/nr
```

or

```shell
yarn add -D @siketyan/nr
```

## Usage

### Run in serial (default)

```shell
nr foo bar baz
```

### Run in parallel

```shell
nr -p foo bar baz
```
